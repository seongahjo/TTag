# TTag

 태그 추출 라이브러리
 
 문서를 업로드하면 파일의 내용을 분석하여 태그를 만들어주는 라이브러리
 
 Extract Tag Library
 
 When you upload a file then this library helps you to make tags which are related to its content
 
 
Supported Document Type
=====
* txt
* Doc
* Docx
